Data Science Project

World Cup Predictor

A FIFA World Cup 2026 prediction system built with large-scale Monte Carlo simulation to estimate each team’s probability of reaching every stage of the tournament. The model combines an ensemble of XGBoost, LightGBM, and Logistic Regression trained on international match data from 2012 to 2025, using features such as ELO rating, squad strength, recent form, and goal statistics, then integrates domain-based priors to improve prediction stability across full 48-team tournament simulations.

Tech Stack

Python Monte Carlo Simulation XGBoost LightGBM Logistic Regression ELO Rating Statistical Modeling
On Progress

Anti Spoofing Face Model

A face anti-spoofing classification project built to distinguish real faces from six spoofing categories under real-world variation such as lighting, pose, and subtle visual artifacts. The model uses a dual-stream architecture that combines DINOv2 for global semantic understanding and ConvNeXtV2 with Laplacian high-frequency filtering for local texture analysis, fused through an adaptive gated mechanism for stronger generalization and more stable predictions.
